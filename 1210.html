JavaScript内置对象_Date和BOM对象

Date的声明方式

var time=new Date();

console.log(time);

//"Sat Dec 10 2016 11:02:28 GMT+0800 (中国标准时间)";

var time=new Date(你可以可以直接在这输入日期，如：2016-12-12)

//如何获取年月日等

var timeY=time.getFullYear()//年

var timeM=time.getMonth()+1//月 这里加1是因为月份从0开始，例如11月份它显示的就是10，所以要加1

var timeD=time.getDate()//日 获取日期几号 如果你是自己写的日期，例如2016-12-0，那你获取到的是这个月的天数

var timed=time.getDay()//周几 从0开始，在国外0 代表星期日，所以如果你想要对应，你可以设置数组来对应

var timeH=time.getHours()//小时 0-23

var timem=time.getMinutes()//分钟 0-59

var timeS=time.getSeconds()//秒 0-59

//其他还有毫秒等，自己可以搜索一下

4、History

history.length 返回浏览器历史中列表中的URL数量

history.back() 加载 history 列表中的前一个 URL

history.forward() 加载 history 列表中的下一个 URL

history.go() 加载 history 列表中的某个具体页面

    function goBack()
  {
  window.history.go(-1)     //会返回到前一个页，如果写1会跳转到下一个页面
  }

5、Location

location.host 设置或返回主机名和当前 URL 的端口号

document.write(location.host) 主机名127.0.0.1 端口号8020

location.hostname 设置或返回当前 URL 的主机名

location.port 设置或返回当前 URL 的端口号

document.write(location.port) 当前端口号8020

下午测试题

用Array自带的排序方法对

	var arr = [100,40,2,101,20,25,49,4];
	console.log("排序前："+arr);
	arr.sort(function(a,b){
		return b-a;
	});
	console.log("排序后："+arr)
	"排序前：100,40,2,101,2,0,25,49,4"
	"排序后：101,100,49,40,25,20,4,2"
	这个是降序，如果写成a-b则是升序
