JavaScript函数默认返回值是什么

undefined

			var a=1;
			function uu(){
				a++;
				alert(a);
				var a=2;
				alert(a);
			}
			uu();

第一个a是NaN,第二个a是2

虽然你在外部声明了a，但它会先执行函数内部的语句。a++就等于a=a+1，因为你一开始没给a赋值，+1就等于两个不同类型的值相加，所以会出现NaN

第二个因为先执行函数内部的，有了声明也给了赋值所以直接等于2

var f=function g(){

会报错：ReferenceError: g is not defined

理解为将g函数的值赋值给了f，g函数就类似于在一个函数里面，也就是所谓的闭包，只是类似不是一样，所以在外部调用这个函数是没用的，所以会报错
