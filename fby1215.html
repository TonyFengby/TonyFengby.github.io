用Canvas画猫

1、先想好要画的步骤，因为在最后上色的时候上层的会将下层的颜色遮住

2、首先画猫的脸，用两次三次方贝塞尔曲线bezierCurveTo()来完成

每一次画一半，第一次的终点就是第二次的起点，控制它的控制点来调整形状

3、画眼睛，可以用圆也可以用贝塞尔曲线，圆就是比较规矩一点，看个人喜欢，但要注意不要忘记眼白

4、接着鼻子，同样是用圆跟贝塞尔曲线

5、两边的胡须直接用直线来完成

6、耳朵也用贝塞尔曲线

7、接着画爱心，同样用两次贝塞尔曲线

8、身体可以从左边开始也可以从右边开始，我从右边开始先用贝塞尔曲线画完右边的脚

9、再画一条直线，最后再用贝塞尔曲线完成左脚

10、最后还是用贝塞尔曲线来完成尾巴

XML的简单了解

			
XML 指可扩展标记语言（EXtensible Markup Language） 
			
XML 是一种标记语言，很类似 HTML 
			
XML 的设计宗旨是传输数据，而非显示数据 
			
XML 标签没有被预定义。您需要自行定义标签。 
			
XML 被设计为具有自我描述性。 
			
XML 是 W3C 的推荐标准 
			
具体的用途及操作请详见官方手册
