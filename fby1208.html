JavaScript的内置对象

1、JavaScript的内置对象_String

String对象

	new String(s);
	String(s)
	s 是你要转换的原字符串的值.

String对象属性

	var str="这是String对象属性"
	console.log(str.length());
	//会显示它的长度是12;

String对象方法

indexOf()返回某个指定的字符串值在字符串中首次出现的位置/p> 

	var str = "我是字符我串";
	console.log(str.indexOf("我",str.indexOf("我")+1));
	//会显示4,如果你搜不到你要找的会输出-1,这种方法比较安全,+1是可以搜索第二个位置

match()字符串内检索指定的值，或找到一个或多个正则表达式的匹配

	var str = "我是字符我串";
	str.match("是");
	//如果匹配到则返回值,否则输出null;

substr在字符串中抽取从 start 下标开始的指定数目的字符

	var str = "我是字符我串";
	str.substr(1,2);
	//1代表从第二个字符开始截取,截两个字符,不改变原字符串;

slice提取字符串的某个部分，并以新的字符串返回被提取的部分

split把一个字符串分割成字符串数组

	var str = "我是字符我串";
	console.log(str.split("是"));
	//会显示    "我是,字符我串,"  返回的两个数组;

3、JavaScript的内置对象_Object

Object对象方法

isPrototypeOf(object):判断该对象是否为另一个对象的原型

hasOwnProperty(property):判断对象是否有某个特定的属性。必须用字符串指定该属性（例如，o.hasOwnProperty(”name”)）
